# QR Studio - Premium QR Code Generator

A modern, high-performance QR code generator built with React and Vite.

## Features

-   **Gradient Support**: Create stunning QR codes with linear gradients.
-   **Custom Shapes**: Choose between dots, squares, and rounded styles.
-   **High Quality Export**: Download as PNG for professional use.
-   **Dark Mode**: Sleek, glassmorphism-inspired interface.

## Local Development

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the dev server:
    ```bash
    npm run dev
    ```

## Deployment with Dockploy

This project is ready for deployment using [Dockploy](https://dockploy.com/).

1.  **Push to GitHub**: Commit your changes and push this project to a GitHub repository.
2.  **Create Application in Dockploy**:
    -   Go to your Dockploy dashboard.
    -   Select your server.
    -   Click "New Application".
    -   Select "Git" as the source.
    -   Choose your repository.
3.  **Configuration**:
    -   Dockploy will automatically detect the `Dockerfile`.
    -   Ensure the port is set to `80` (as exposed in the Dockerfile).
    -   Click "Deploy".

Your app will be built and served automatically!
